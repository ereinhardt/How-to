# How To 

**FFMPEG IS MANDATORY**

## Convert 

```bash
npm i
npm run build 
npm run convert dir=<dirname>
```


## Config 
For the config you need a .env file following fields: 

```env
GEMINI_API_KEY=<>
USER_QUESTION_INDEX=<>
SERVER_HOST=<>
SERVER_PORT=<>
USERS_FOLDER=<>
USER_FOLDER_ENDING=<>
USER_STREAM_FILE_ENDING=<>
DUMMY_ID=<DEBUG>
```
